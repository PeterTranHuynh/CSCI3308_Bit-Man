Detailed Description

Information about the viewing angles through the lenses.

All angles in degrees, measured away from the optical axis, i.e. angles are all positive. It is assumed that left and right eye FOVs are mirror images, so that both have the same inner and outer angles. Angles do not need to account for the limits due to screen size.
